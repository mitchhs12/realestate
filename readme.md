# Viva Ideal

The following repository is the backend and the frontend of Viva Ideal.
A property marketplace for real-estate located in Latin America targeted at international buyers.

The website features the following:
Integration with the Google Translation API, Google Maps Locations API, Google Maps Places API.
Integration with Replicate AI API (for using their cloud-based AI image models).
Integration with the Stripe API for handling of subscriptions and payments.
Integration with AWS Locations API for search suggestions.
Integration with AWS S3.
Integration with Upstash for Redis.
Integration with Neon PostgreSQL database for storage
